Good. Now we define the **Builder Flow**
This is the step by step path the user walks through to create a resume.

This flow is what makes the app feel simple instead of overwhelming.

Pure text.
Concept only.
No code.
No symbols.

---

BUILDER FLOW OVERVIEW

User always moves through the resume in a logical order

Welcome
Template Selection
Profile
Summary
Experience
Education
Skills
Projects
Optional Sections
Review
Export

They can go forward or backward at any time.

Progress is auto saved.

---

STEP ONE
WELCOME STEP

Purpose
Make the user start immediately

What they see
Create Your Resume
Choose a format to begin

Actions
Start New Resume
Continue Existing Resume

System behavior
Creates a new empty resume object
Sends user to template selection

---

STEP TWO
TEMPLATE SELECTION

Purpose
Let user visually choose style

What they see
Grid of resume formats
Each shows name and preview

Example names
Classic One
Modern Prime
Executive Two
Creative Wave
Tech Matrix
Minimal Zen

Actions
Click a format
Preview larger
Select and continue

System behavior
Saves chosen format
Moves to profile step

---

STEP THREE
PROFILE STEP

Purpose
Collect identity information

Fields shown
Full Name
Professional Title
Email
Phone
City
State
Country
LinkedIn
Portfolio

Optional
Profile photo

User experience
Live preview updates instantly
Validation checks
Email must look like email
Phone must look like phone

Continue button
Next to Summary

---

STEP FOUR
SUMMARY STEP

Purpose
Create professional overview

Fields shown
Headline optional
Summary text

Helper text
Two to four sentences recommended

Preview behavior
Summary appears at top of resume

Buttons
Back
Next

STEP FIVE
EXPERIENCE STEP

Purpose
Build job history

User actions

Edit job
Delete job
Reorder jobs

Job Title
Company Name
Location
Start Date
End Date
Current job toggle
Description bullets

Guidance text
Use action verbs
List achievements

System behavior
Experience sorted newest first

Delete education

Current study toggle

Optional

Grade

Skill level
