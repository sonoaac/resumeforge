RESUME DATA STRUCTURE
Single Resume Object

A resume is made of these main parts

Profile
Summary
Experience
Education
Skills
Projects
Certifications
Awards
Languages
References
Custom Sections

Each part has its own internal fields.

---

PROFILE SECTION

This is the identity of the resume.

Full Name
Professional Title
Email
Phone Number
City
State
Country
LinkedIn Link
Portfolio Link
Website Link

Optional
Profile Photo
Nationality

---

SUMMARY SECTION

This is a short paragraph or two.

Summary Text
Optional Headline
Optional Career Goal

Rules
Maximum four sentences
No bullet points
Plain text only

---

EXPERIENCE SECTION

This is a repeating group.
User can add unlimited jobs.

Each job contains

Job Title
Company Name
Company Location
Start Date
End Date
Is Current Job true or false

Description Bullets
Bullet One
Bullet Two
Bullet Three
Bullet Four
Bullet Five

Optional
Industry
Employment Type
Achievements

---

EDUCATION SECTION

This is also repeating.

Each education entry contains

Degree
Field of Study
School Name
School Location
Start Date
End Date
Is Current Study true or false

Optional
Grade
Honors
Relevant Coursework

---

SKILLS SECTION

This is a list group.

Each skill contains

Skill Name
Skill Level
Beginner
Intermediate
Advanced
Expert

Optional
Years of Experience
Category such as Technical or Soft

---

PROJECTS SECTION

This is repeating.

Each project contains

Project Name
Project Role
Project Description
Start Date
End Date
Tools Used
Project Link

Optional
Team Size
Outcome

---

CERTIFICATIONS SECTION

Repeating entries.

Each certification contains

Certification Name
Issuing Organization
Issue Date
Expiration Date
Credential Link

---

AWARDS SECTION

Repeating entries.

Each award contains

Award Name
Awarding Body
Date Received
Description

---

LANGUAGES SECTION

Repeating entries.

Each language contains

Language Name
Proficiency
Basic
Conversational
Fluent
Native

---

REFERENCES SECTION

Repeating entries.

Each reference contains

Reference Name
Company
Position
Email
Phone

Visibility Rule
Hidden by default
Only shown if user enables

Section Title
Entries

Entry Title
Entry Description
Entry Dates

Examples

Military Service
Leadership

---

SECTION ORDER CONTROL

Formats can change the order but never the structure.

Possible orders

Profile
Summary
Experience
Education
Skills
Projects
Certifications
Awards
Languages
References
Custom Sections

Example
Modern format may put Skills above Experience
Classic format puts Experience first

Data stays the same.
Only layout changes.
